== Description ==

MozBot Chat Widget brings the power of AI-driven customer support directly to your WordPress website. With our easy-to-use plugin, you can deploy a fully customizable chatbot that provides instant responses to your visitors' questions, captures leads, and improves customer engagement.

= Key Features =

* **AI-Powered Responses** - Intelligent chatbot that understands and responds to customer queries
* **Full Customization** - Match your brand colors, customize messages, and position the widget anywhere
* **Multi-Channel Support** - Extend conversations to Telegram, WhatsApp, Messenger, and more
* **Automation Workflows** - Connect with n8n and other automation tools for advanced workflows
* **Analytics & Reporting** - Track conversations, response rates, and customer satisfaction
* **Easy Setup** - Get started in minutes with our simple configuration interface
* **Responsive Design** - Works perfectly on desktop and mobile devices
* **24/7 Support** - Round-the-clock customer assistance

== Installation ==

= Automatic Installation =

1. Log in to your WordPress admin panel
2. Go to Plugins > Add New
3. Search for "MozBot Chat Widget"
4. Click "Install Now" and then "Activate"

= Manual Installation =

1. Download the plugin zip file
2. Upload it to your `/wp-content/plugins/` directory
3. Unzip the file
4. Activate the plugin through the 'Plugins' menu in WordPress

= Configuration =

1. Go to MozBot in your WordPress admin menu
2. Enter your Bot ID from your MozBot dashboard
3. Customize the appearance and settings
4. Enable the widget and save your settings

== Frequently Asked Questions ==

= Do I need a MozBot account to use this plugin? =

Yes, you need to create a free account at MozBot.com to get your Bot ID and access the chatbot functionality.

= Is the plugin free? =

The plugin is free to install and use. MozBot offers both free and paid plans depending on your needs and usage.

= Can I customize the appearance of the chat widget? =

Absolutely! You can customize colors, position, messages, and even add custom CSS to match your brand perfectly.

= Does the widget work on mobile devices? =

Yes, the chat widget is fully responsive and works seamlessly on all devices including smartphones and tablets.

= Can I integrate with other tools? =

Yes, MozBot supports integrations with n8n, Zapier, and many other automation tools to create powerful workflows.

= How do I get support? =

You can reach our support team through the MozBot dashboard, or visit our documentation at docs.mozbot.com.

= Can I use this on multiple websites? =

Yes, you can use MozBot on multiple websites. Each site can have its own chatbot configuration.

= Is my data secure? =

Yes, we take data security seriously. All conversations are encrypted and stored securely. We're GDPR compliant.
